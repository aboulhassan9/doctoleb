# PDF/A Assets

These assets are bundled with `render-clinical-document` so PDF/A-2b output does not depend on system fonts or runtime network access.

- `NotoSans-Regular.ttf` and `NotoSans-Bold.ttf`: Google Noto Sans, SIL Open Font License.
- `sRGB.icc`: ICC sRGB v2 profile from the public ICC registry, used as the PDF/A output intent.
